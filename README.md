# Árvore do Conhecimento

Hub central do GitHub **CanonEngineer** — cada projeto é uma árvore interativa com nós de código.

## Projetos

| Projeto | Stack |
|---------|-------|
| CanonPythonEcommerce | Django 5 |
| FoodOnlineDjango | Django + PostGIS + RazorPay |
| JavaScriptDropBoxProject | Express + Firebase |
| JavaScriptUsersProject | ES6 + localStorage |
| JavaScriptRestfulApiProject | Express + NeDB |
| Professional-Scanner | Node.js + WebSocket |
| CustomizeVeyonProject | Qt/C++ Veyon CIMED |

## Uso

Abra `index.html` ou publique via GitHub Pages.

- **Hover** no nó → nome e descrição
- **Clique** no nó → modal com código e implementação
- **Fechar** → volta à árvore
